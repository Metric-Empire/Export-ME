Export ME - Blender Extension Repository

Export ME is a Blender add-on for exporting FBX files to game engines, with automated mesh processing, collider handling, material management, and project-based export workflows.

Installation via Extensions Platform

  1. Go to Blender Preferences → Get Extensions → Repositories → ＋ → Add Remote Repository
  2. Set the URL to: https://metric-empire.github.io/export-me/index.json
  3. Enable Check for Updates on Start
  4. Filter the available extensions for "Export ME" and install

Installation via Drag-and-Drop

Click and drag the file link from the table below into Blender to install directly.

Installation via Download

Download the .zip file and drag-and-drop it into Blender, or install via Preferences → Get Extensions → Install from Disk.

Note: Manual installation methods will not connect to this repository and updates will not be automatically available.
